# ArgentBank - Projet 10

## Description

Ce projet fait partie de ma formation d'intégrateur web chez OpenClassrooms.  
Cette mission consiste à développer le frontend d’une application bancaire fictive, ArgentBank, en utilisant React et Vite à partir d’un backend existant fourni par l’école.
Le travail comprend à la fois l’intégration des pages et de la navigation avec React et React Router, ainsi que la mise en place de la gestion de la connexion utilisateur et l’utilisation de Redux pour gérer les données de l'application.

## Compétences visées

- Initialiser un projet React avec Vite
- Créer des composants réutilisables
- Gerer la navigation avec React Router
- Intégrer une API REST
- Gerer l’état global avec Redux
- Sécuriser des routes et gestion de la session utilisateur
- Utiliser les outils de développement React (DevTools, Redux DevTools)
- Versionner le code avec Git et GitHub

## Technologies

- React (Vite)
- Redux Toolkit
- React Router
- HTML5 / CSS3 / SCSS
- JavaScript (ES6+)
- Node.js
- Git / GitHub

## Pré-requis

- NodeJS >= v12
- NPM
- Le backend d’ArgentBank doit être installé et lancé localement [(Voir le dépôt backend)](https://github.com/Jonathan-Huber/ArgentBank-Backend)

## Installation

1. Cloner le projet :

```bash
git clone https://github.com/Jonathan-Huber/ArgentBank-Frontend.git
```

2. Installer les dépendances :

```bash
npm install
```

## Lancement de l'application

```bash
npm run dev
```
